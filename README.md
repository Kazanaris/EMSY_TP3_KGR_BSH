# EMSY_TP3_KGR_BSH
Repo du Git pout le TP3 d'EMSY


## convertisseur de données 
### mode d'emploi
+ monnaie disponible
les monnaies disponible sont les suivantes : 
  * Francs suisse / CHF
  * Réal brésilien / BRL
  * Peso Mexicain / MXN
  * Couronne norvégienne / NOK
+ choix de la devise à convertir
pour choisir la devise a convertire, il faut choisir dans la liste les 4 pays disponible :
